# FoodGo - React Native Food App

FoodGo is a simple food ordering mobile application built with React Native and Expo. I created this project while practicing mobile app development, focusing on screen navigation, UI design, reusable screens, and basic cart functionality.

## Project Purpose

The purpose of this project was to strengthen my understanding of React Native by building a practical mobile app with multiple screens and user flow.

## Features

- Onboarding screen
- Login screen
- Registration screen
- Forgot password screen
- Home screen
- Food details screen
- Cart screen
- Add to cart functionality
- Basic navigation between screens
- Mobile-friendly user interface

## How to Run the Project

Clone the repository:

```bash
git clone https://github.com/muna-jemima/FoodGo-React-Native.git
```

Go into the project folder:

```bash
cd FoodGo-React-Native
```

Install dependencies:

```bash
npm install
```

Start the project:

```bash
npx expo start
```
